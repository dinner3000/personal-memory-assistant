# Roadmap

## Phase 1: Foundation ✅ *(current)*

- [x] Project structure and documentation
- [x] Git repo + GitHub remote
- [x] Journal entry format defined
- [x] Hermes skill for journal recording
- [x] Hermes skill for journal retrieval
- [ ] First real journal entries recorded

## Phase 2: Automation

- [x] `new-entry.sh` — create formatted entry from a prompt
- [x] `search.sh` — full-text search across journal
- [ ] `monthly-digest.sh` — aggregate month into summary
- [ ] `backup.sh` — archive to tar.gz or remote

## Phase 3: Hermes Skill Integration

- [ ] Skill: `journal-record` — I write entries when you tell me things
- [ ] Skill: `journal-retrieve` — I search + summarize when you ask
- [ ] Skill: `journal-digest` — I generate periodic summaries proactively
- [ ] Register skills in the Hermes skills hub

## Phase 4: Advanced Retrieval

- [ ] Semantic search (optional — embeddings + vector search)
- [ ] Tag-based filtering and aggregation
- [ ] Timeline view generator (render journal as a visual timeline)
- [ ] Cross-reference and relationship mapping

## Phase 5: Multi-Device Sync

- [ ] git auto-push/pull for journal sync across machines
- [ ] Cron job for periodic commit + push
- [ ] Mobile access (via Obsidian sync or Hermes gateway)

## Phase 6: Proactive Intelligence

- [ ] Weekly summary pushed to a configurable destination (Telegram/email)
- [ ] On-this-day recall ("A year ago today you wrote about...")
- [ ] Pattern detection (recurring topics, mood trends, habit tracking)
- [ ] Annual report — generate a "year in review" from journal entries
